id: EDQA201501CREGL2556808  --EDQid de un libro

EDQ son el nombre del negocio; A es el tipo de producto, como es A es un "Libro"; 201 corresponden a los ultimos 3 digitos del id del proveedor del producto, el id se asignó previamente al proveedor; 501 corresponden a los ultimos tres digitos del id de la editorial del producto en caso de que tenga una, la id de la editorial igualmente fue anteriormente asignada; CR corresponden a un código asignado al género y EG igualmente respecto al subgénero; L, es lo que define el "kind" del producto, si es Line (L) u Outlet (O); los últimos 7 digitos son aleatorios;
al crearse un id cuando se crea un producto en la base de datos debe validarse si el edqid ya existe, si es así, los últimos 7 digitos aleatorios deben cambiar

id: EDQB101RA2556808   --EDQid de un separador
EDQ; B es el tipo, Separador; 101, es el proveedor; RA, es el género; 7 números aleatorios
